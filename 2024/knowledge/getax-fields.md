# GeTax 2024 - Référence Complète des Rubriques

Ce document contient TOUTES les rubriques GeTax avec leurs codes exacts pour guider le contribuable pas à pas.

---

## Convention de numérotation

- **1x.xx** = Revenus Contribuable A
- **2x.xx** = Revenus Contribuable B
- **3x.xx** = Déductions Contribuable A
- **4x.xx** = Déductions Contribuable B
- **5x.xx** = Déductions générales (prévoyance, assurances)
- **6x.xx** = Fortune mobilière
- **7x.xx** = Fortune immobilière
- **8x.xx** = Charges de famille
- **9x.xx** = Récapitulatifs

## Exemples de calcul

### Exemple 1: Forfait frais professionnels

```
Données:
- Salaire brut (11.10): 100'000 CHF
- Cotisations AVS (31.10): 5'300 CHF
- Cotisations LPP (31.12): 7'000 CHF

Calcul:
Base = 100'000 - 5'300 - 7'000 = 87'700 CHF
Forfait 3% = 2'631 CHF

→ ICC (31.50): 1'796 CHF (plafonné)
→ IFD (31.20): 2'631 CHF (dans limites)
```

### Exemple 2: Frais de repas (ICC)

```
Conditions: Trajet TP > 2h/jour
Jours travaillés: 220 jours
Case G: Non cochée

Calcul:
220 × 15 CHF = 3'300 CHF

→ 31.60: 3'200 CHF (plafonné)
```

### Exemple 3: Déduction 3b (assurance-vie)

```
Situation: Couple marié, 2 enfants, les deux affiliés LPP

Limites ICC:
- Couple: 3'486 CHF
- Par enfant: 2 × 951 = 1'902 CHF
- Total max: 5'388 CHF

Prime effective: 4'000 CHF
→ 52.15: 4'000 CHF (dans limites)
```

## Checklist de saisie GeTax

### Avant de commencer
- [ ] Certificat(s) de salaire complet(s)
- [ ] Attestation 3e pilier A (formulaire 21 EDP)
- [ ] Attestation rachats LPP
- [ ] Relevés bancaires au 31.12
- [ ] État des titres
- [ ] Factures frais de garde
- [ ] Attestation assurance maladie

### Ordre de saisie recommandé
1. [ ] Situation personnelle (état civil, enfants, commune)
2. [ ] Revenus (11.xx/21.xx)
3. [ ] Cotisations sociales (31.10/41.10, 31.12/41.12)
4. [ ] Prévoyance (31.30/41.30, 31.40/41.40)
5. [ ] Frais professionnels (31.20 ou 31.60-31.71)
6. [ ] Fortune (60.xx, 65.xx)
7. [ ] Dettes (55.xx, 66.xx)
8. [ ] Assurances (52.xx)
9. [ ] Autres déductions (53.xx, 59.xx)
10. [ ] Vérification et signature
