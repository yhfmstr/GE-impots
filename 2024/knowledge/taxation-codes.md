## Utilisation des codes

### Adresse de résidence
- Indiquer le code commune (6601-6645) pour Genève

### Lieu de travail
- Même commune: code de la commune
- Autre canton: indiquer code du domicile (exception)
- Exemple: Domicile Versoix (6644), travail Nyon → Code 6644

### Biens immobiliers
- Genève: code commune
- Autre canton: code canton (9001-9026)
- Étranger: code pays (8xxx)

### Comptes bancaires étrangers
- Utiliser code pays correspondant
